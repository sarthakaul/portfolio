#7 Audi Web App

Hey guys, this is a web app that me, Mohammed, and Brian created as a team. It is a web shop that allows users to buy cars, check orders, and wishlist items. The original design took about 2 months, and most of the implementation and updated changes happened in January 2023. This website uses some ASP.NET Core technologies such as Razor Pages, and Entity Framework. The website is dynamic because it is provided unique functionality by combining C# and HTML in Razor Pages. CRUD pages are scaffolded automatically with Entity Framework, allowing the user to create, read, update, delete, page information. It also features an SQL Server database where records can be modified from these CRUD pages. There are 6 CRUD pages (all of which were seeded with 10 records each in the new repository).

Language: C#, HTML
Frameworks: ASP.NET Core, Entity Framework, Bootstrap
