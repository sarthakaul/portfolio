Hey guys, this is a real-time system that was designed to help SMEs (small businesses) reach Net Zero. I started this in July 2023 and it took a few days to finish. It was done as part of a Technology internship which involved doing industry research, also it was largely creative for participants. I considered that minimising Scope 2 emissions with this real-time system would be beneficial, and possible in the real world - although it would incur some initial expense. Top building energy emissions source are heating, lighting, and computers, so this real-time system manages all three. It keeps lights and computers off when people are not in an area, and will activate them and turn heating on for employees if they are detected. It does this by using environmental sensors (heat, light, motion) to signal a control unit, which then communicates wirelessly with smart switches to provide power to connected lights, wake computers, activate air conditioners. The system operates automatically, and continuously. Administrators and technicians can remotely activate building facilities ICEMS manages through an app or web dashboard.

I created a state machine model for this project with some wireframes. The state machine model includes a state machine diagram, state and stimuli chart (not shown here), 3 sub-state diagrams (which I completed after the work was submitted because I had a maximum slide count, but felt involved regardless), 3 thread diagrams, and a couple of screen wireframes (these were shown on the presentation I sent). There are more aspects to a finished real-time system such as architectural design, test specifications, policies, documentation, and plans to name a few.
