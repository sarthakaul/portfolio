#8 Accounts and Tickets Program

Hey guys, this is a program that manages mock bank accounts and theatre ticket seats. The program took a few weeks overall to develop starting April 2023. It uses record data structures to store account names and balances, and functions to check account balances, then sort some accounts according to a balance threshold. It also generates some ticket records with seat numbers. It can lock function data to prevent a race condition using threads. It runs through a console to display and accept information from the user.

Language: F#
Framework: .NET
IDE: Visual Studio
